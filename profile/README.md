# 🖥️ Harddex - Hardware Indexer & Comparison Platform

> Uma plataforma moderna e robusta para catalogar, buscar e comparar componentes de hardware de computadores, funcionando como uma "Pokedex de Hardware".

---

### 🚀 Acesse o Repositório Principal

Este repositório faz parte da organização **Harddex**. Para acessar o código-fonte atualizado, documentação completa de arquitetura, guias de setup das camadas (API e Frontend Web) e acompanhar as sprints de desenvolvimento da equipe, acesse o nosso repositório oficial:

👉 **[https://github.com/ammvy/harddex-landing](https://github.com/ammvy/harddex-landing)**

---

### 🛠️ Tecnologias de Destaque no Projeto
*   **Monorepo**: Gerenciado com PNPM Workspaces.
*   **Backend (API)**: Fastify, TypeScript, Drizzle ORM, Zod, PostgreSQL (Docker) e Firebase Admin SDK.
*   **Frontend (Web)**: Next.js 16 (App Router), React 19, Tailwind CSS v4, Motion e Radix UI.
