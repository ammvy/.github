# Hardex

Bem-vindo ao repositório do projeto **Hardex**, desenvolvido durante o semestre 1° semestre/2026. Este espaço foi organizado para demonstrar o progresso, a colaboração e as entregas técnicas da equipe.

### 🎨 Paleta de Cores (Regra 60/30/10)

Para um projeto de hardware que quer ser acessível, precisamos equilibrar a energia do Crimson com cores neutras que passem confiança e facilitem a leitura de dados técnicos.

---

#### ⚪ **60% - Cor Primária (Fundo/Base): Off-White Tech**
- **Hex:** `#F4F4F9`
- **Uso:** Fundo do site/app, áreas de conteúdo principal. É um branco levemente acinzentado e frio, que cansa menos os olhos que o branco puro e remete a um ambiente limpo e tecnológico.

#### ⚫ **30% - Cor Secundária (Texto/Estrutura): Deep Charcoal**
- **Hex:** `#2B2D42`
- **Uso:** Textos principais, títulos, bordas de tabelas, ícones de navegação. É um cinza muito escuro, quase azulado, que oferece excelente contraste e leitura, sem o peso do preto puro.

#### 🔴 **10% - Cor de Destaque (Accent): Vibrant Crimson**
- **Hex:** `#D62828`
- **Uso:** Botões de "Comparar", notas de hardware (as notas altas), links ativos, o ícone principal e elementos que precisam de atenção imediata (como a barra de busca).

---

**Por que essa combinação?**
O fundo claro (`#F4F4F9`) mantém o visual "acadêmico" e limpo. O texto escuro (`#2B2D42`) garante a legibilidade. O Crimson (`#D62828`) explode contra essas cores neutras, guiando o usuário para as ações principais, assim como a Pokédex destaca as informações do Pokémon.

## 📌 Objetivo do Projeto

O Hardex é uma plataforma de gerenciamento de hardware que visa facilitar a vida de técnicos e usuários de celulares e notebooks. Ele permite o cadastro, edição, exclusão e visualização de informações sobre hardwares, como processadores, placas-mãe, memórias RAM, placas de vídeo, HDs, SSDs, fontes, gabinetes e periféricos.
